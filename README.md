## 🌟 Main Project

### Noorify Bot

<div align="center">

[![View Code](https://img.shields.io/badge/View_Code-6D28D9?style=for-the-badge&logo=github&logoColor=white)](https://github.com/RamiAIlab/Noorify_Bot)
[![Try Bot](https://img.shields.io/badge/Try_It-C026D3?style=for-the-badge&logo=telegram&logoColor=white)](https://t.me/Noorify_bot)
[![Open Source](https://img.shields.io/badge/MIT_License-F472B6?style=for-the-badge&logo=opensourceinitiative)](https://github.com/RamiAIlab/Noorify_Bot/blob/main/LICENSE)

</div>

**What It Does:**
Daily Telegram bot that sends reminders and handles automation. Built with Python and deployed on Railway.

**Technical Details:**
- Python 3.9+ with async/await
- python-telegram-bot library
- SQLite for data storage
- Task scheduling system
- Deployed on Railway cloud platform
- Error handling and logging included

**Key Features:**
```
✓ Daily automated reminders
✓ User preference system
✓ Reliable scheduling
✓ Cloud deployment
✓ Production-ready code
```

**Performance:**
- Response time: ~100ms
- Uptime: 99%+
- Handles 500+ users
- Minimal resource usage
